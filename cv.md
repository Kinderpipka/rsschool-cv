## Примеры кода

### Задача

Напишите функцию, которая будет принимать два целых числа и возвращать сумму всех целых чисел в этом диапазоне, включая границы.


def sum_of_range(start, end):
    
    if start > end:
        start, end = end, start  

    total_sum = 0
    for num in range(start, end + 1):
        total_sum += num
        
    return total_sum
